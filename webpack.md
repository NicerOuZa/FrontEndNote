# 一，安装和配置

### 1，安装

```js
//全局安装
npm install -g webpack
//安装到你的项目目录
npm install --save -dev webpack
//安装webpack-cli
npm install --save -dev webpack-cli
```

### 2，配置

**0配置**

```js
-打包（支持js的模块化）
npx webpack
```

**手动配置webpack**

默认配置文件的名字  `webpack.config.js`

```js

```
